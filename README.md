#  ElectroStore App

ElectroStore es una aplicación móvil desarrollada con **Flutter**, diseñada para la gestión y venta de productos electrónicos de seguridad.  
Incluye funciones para **clientes** y un **panel administrativo** para la gestión completa del catálogo, pedidos, usuarios y más.

---

##  Características principales

###  **Autenticación y usuarios**
- Registro e inicio de sesión.
- Sistema de **roles**: Cliente y Administrador.
- Persistencia de sesión con tokens seguros.
- Gestión de direcciones del usuario.

###  **Funciones para Clientes**
- Navegación completa de productos.
- Búsqueda avanzada por nombre, categoría o SKU.
- Carrito de compras persistente.
- Procesamiento de pedidos.
- Historial de compras.
- Ver ofertas activas.
- Notificaciones visuales dentro de la App.

###  **Funciones para Administradores**
- Panel administrativo integrado en la misma app.
- Gestión de productos (CRUD).
- Gestión de categorías.
- Gestión de usuarios.
- Control y actualización del estado de pedidos.
- Vista de reportes básicos.

###  **Pagos**
- Integración con **Mercado Pago**.
- Soporte para tarjetas y métodos digitales (incluido Yape mediante Checkout).
- Validación de pedidos y almacenamiento de transacciones.
